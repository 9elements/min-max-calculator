> Made with [Astro](https://astro.build), [Svelte](https://svelte.dev) and [PostCSS](https://postcss.org).

## Local setup

### Install dependencies

```sh
yarn install
```

### Start server

```sh
yarn dev
```

> Done! the server is accessible at http://localhost:3000

## Deployment

```sh
yarn build
```

The output will be in the `dist` folder.
