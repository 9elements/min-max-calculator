# Min-Max-Calculator

Made with [Astro](https://astro.build), [React](https://reactjs.org) and [PostCSS](https://postcss.org).

## Local setup

### Install dependencies

```sh
  yarn install
```

### Start server

```sh
  yarn dev
```

> Done! the server is accessible at http://localhost:3000

## Deployment

```sh
  yarn build
```
